# 실무 콘텐츠 표준 (2단계)

## 필수 원칙

1. AI 결과 ≠ 제어 명령. 안전 제어에 AI 직결 금지.
2. 작물·축종 기준값 임의 확정 금지.
3. PLC 제조사 주소·레지스터 임의 작성 금지.
4. 실행/컴파일하지 않은 코드를 runnable/compileValidated로 표시 금지.
5. 체크/저장이 없으면 저장된 것처럼 표시 금지.

## 실무 페이지 권장 섹션

쉬운 설명 → 원리 → 구성요소/흐름 → 설치·배선 → 설정·스케일 → 계산/공식 → 진단 → 안전 → 유지관리 → 제조사 확인 → 현장 검증 → 출처
