# 콘텐츠 로드맵

## 1단계 (완료 대상)

- 입문·기초 중심 전체 메뉴 콘텐츠
- 교육용 사례, 용어, 프롬프트, 검색·필터
- 데이터 모델 필드에 실무·전문가 확장 여지 포함

## 2단계

- 실무 제어·통신·운영 심화
- 고장진단 절차 확장
- 데이터 품질/알람 우선순위 실무 예시
- 현장 검증(fieldValidationRequired) 항목 축소

## 3단계

- 전문가 분석 예제와 모델 평가 템플릿
- 공식 출처 verified 비율 확대
- 제조사 매뉴얼 연결(manufacturerManualRequired) 정리
