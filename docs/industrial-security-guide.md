# 산업제어 보안 가이드 (농장, 교육용)

## 범위

PLC·게이트웨이·운영 PC·원격접속·데이터·카메라·**제어 명령**·설정·백업을 보호 대상으로 둡니다.  
IEC 62443 등 표준은 **개념 참고**용이며, 본 문서는 준수·인증을 주장하지 않습니다.

## 위협 모델 (대표)

| 위협 | 영향 |
|------|------|
| 기본 비밀번호·계정 공유 | 무단 접속·감사 불가 |
| 피싱·USB | 게이트웨이·운영 PC 침투 |
| 불필요 포트 개방 | 원격 스캔·명령 변조 |
| 설정·PLC 프로그램 변조 | 인터록 우회 |
| 명령 재전송 | 의도치 않은 구동 |
| 랜섬웨어 | 가동 중단·복구 지연 |

[2단계](phase2-completion-report.md) 통신·원격 페이지와 함께 위협을 농장망에 매핑하세요.

## 대책 (실무 방향)

1. **망 분리** — OT(제어) / IT(사무) / 게스트 분리
2. **최소 권한** — 역할별 읽기·쓰기·설정 분리
3. **MFA·VPN** — 원격은 VPN + 다요소 인증
4. **방화벽·포트 최소화** — Modbus/원격 HMI 포트 화이트리스트
5. **감사 로그** — 설정·명령·로그인 ([로그](operations-runbook.md) 연계 `expert-logging`)
6. **백업·복구 시험** — PLC·설정·DB 정기 복원 drill
7. **패치·퇴사자 계정** — 게이트웨이·OS·앱, 퇴직 즉시 회수
8. **비상 로컬 운전** — 원격·클라우드 장애 시 현장 HMI·수동

## AI·원격 제어

- AI·클라우드 대시보드는 **모니터링·권고** — 최종 구동은 PLC·인터록
- [AI 모델 평가](ai-model-evaluation-guide.md)의 안전 계층 준수
- 서비스 계정 JSON·토큰을 Git·로그·스크린샷에 **넣지 않음**

## 농장 특수 고려

- 외부 SI·장비 업체 임시 계정 — 기간·권한·회수
- CCTV·모바일 앱 — TLS·인증서 만료
- [변경관리](change-management-guide.md) — PLC·방화벽·AI 모델 배포
