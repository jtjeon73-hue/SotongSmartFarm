# PLC 제어 가이드 (2단계)

## 모드

정지 / 수동 / 자동 / 원격 / 점검 / 장애 / 비상

모드별 허용·금지 명령, 전환·복귀 조건, 기록을 정의한다. 원격은 현장 수동·인터록·E-stop을 덮지 않는다.

시작·정지·인터록·지연·최소운전/정지·피드백·타임아웃·실패·재시도·알람·수동·통신/센서장애·정전복구·비상

예제 코드는 educationalExample / pseudocode로 표시. 현장 적용 전 제조사 매뉴얼·전기설계·시험 필수.
