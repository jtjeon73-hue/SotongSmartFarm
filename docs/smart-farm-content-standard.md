# 스마트팜 콘텐츠 작성 표준

## 필수 원칙

- AI 결과와 제어 명령을 구분한다.
- 작물·가축·시설·안전에 영향을 주는 제어에는 사람 검토, 제한조건, 비상정지, 수동운전을 명시한다.
- 작물 생육 기준값을 공식 자료 없이 임의로 제시하지 않는다.
- PLC 제조사별 주소·명령어를 매뉴얼 확인 없이 작성하지 않는다.
- 실제 구축 실적이 아니면 교육용 설계 사례로 표시한다.

## 콘텐츠 필드

- id, title, summary, category, difficulty, contentType
- farmTypes, sections, keywords, relatedIds, sourceIds
- verificationStatus, checkedAt

## 난이도

- 입문 / 기초 / 실무 / 전문가

## 검증 상태

- verified
- versionDependent
- needsReview
- manufacturerManualRequired
- fieldValidationRequired
- educationalExample
