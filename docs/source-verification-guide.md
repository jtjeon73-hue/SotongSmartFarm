# 출처 검증 가이드

## 우선 출처

- 농촌진흥청, 농림축산식품부, 스마트팜코리아
- 국립농업과학원, 한국농어촌공사
- 한국전기안전공사, 한국산업안전보건공단
- 국가법령정보센터
- PLC/센서 제조사 공식 매뉴얼·데이터시트
- Modbus, Flutter, Firebase, Microsoft 공식 문서

## 상태 규칙

- 직접 확인하지 않은 자료를 `verified`로 표시하지 않는다.
- 지원정책·법령·가격·장비 사양은 변경 가능하므로 고정 사실처럼 쓰지 않는다.
- 교육용 설명은 `educationalExample`을 사용한다.
